BBL 2019-20 Batting Performance Analysis (Excel)
Built a comprehensive Excel-based analytics project analyzing batting performance across the 2019-20 Big Bash League (BBL) season. The dataset covers 50+ players across all franchises, tracking match-level metrics including runs, balls faced, strike rate, batting average, not-outs, boundary counts (4s/6s), and partnership data.
Key deliverables:

Performance ranking system — custom "Performance Points" metric combining strike rate, average, and volume to rank players league-wide
Player-level dashboards — Total Runs, Batting Average, Not Outs, and Highest Scores broken down by individual player
Team-level analysis — team-wise total runs and strike rate comparisons across all BBL franchises (Scorchers, Hurricanes, Stars, Strikers, etc.)
Boundary hitting analysis — dedicated view of most 4s/6s and boundary run contribution, with boundary percentage calculations
Partnership & ranking insights — highest partnerships by wicket, partnering players, and the teams they came against
Interactive dashboard sheet consolidating key visuals for at-a-glance season insights

Techniques used: Advanced Excel formulas (SUMIFS, RANK, calculated fields), pivot-style summary tables, multi-sheet data modeling, and dashboard design for sports performance analytics.
This project demonstrates the ability to structure raw multi-column sports data into decision-ready insights — translating directly into business use cases like sales performance ranking, KPI dashboards, and team/segment comparison reporting
